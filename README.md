# SVM Iris Flower Prediction App

This project is a Machine Learning web application developed using Streamlit and Support Vector Machine (SVM).

The app predicts the species of an Iris flower based on flower measurements entered by the user.

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NumPy

---

## Machine Learning Algorithm

This project uses Support Vector Machine (SVM) Classification.

Different kernels used:
- Linear Kernel
- RBF Kernel
- Polynomial Kernel
- Sigmoid Kernel

---

## Dataset Used

Iris Dataset from Scikit-learn.

The dataset contains:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

---

## Features of the App

- User-friendly interface
- Predict Iris flower species
- Select different SVM kernels
- Display model accuracy
- Real-time prediction
